# citypulse

Citypulse is a project utilizing open-source data to analyze inequality among different neighborhoods and boroughs in New York City. Cleaned CSV files is in the "datasets" directory; code used to clean the open-source data is in the "cleaning datasets" directory; the visualizations made with these datasets are in the "visualizations" directory. Each member of the group wrote their own python script to clean the specific datasets they intended to use. Project by @carstenpetey @nghi-nguyen23 @cindysgit
